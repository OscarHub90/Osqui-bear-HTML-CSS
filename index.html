Estructuras semánticas HTML

<div/> = Etiqueta contenedora que no tiene semántica

<header/> = Cabecera de un sitio web o de una sección
<main/> = Define la sección ppal del documento, solo puede existir una etiqueta main por cada documento

<nav/> = representa una navegación o botonera
<article/> = representa una sección de autocontenido (que se explica por sí sola, ejemplo un artículo en una revista)

<aside/> = representa un contenido complementario o secundario.
<section/> = representa una sección de contenido genérica.
<address/> = Representa direcciones.
